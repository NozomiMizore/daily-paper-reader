## 摘要
从治疗前转录组准确预测患者特异性治疗反应受到匹配临床反应标签和治疗后分子图谱稀缺的阻碍。临床前迁移学习模型可以模拟药物诱导的表达变化，但通常难以解释且不稳定，而知识图谱方法提供了机制背景，但保持静态且无法捕捉药物诱导的转录组扰动动态。我们提出PREDIKTOR，一个以患者为中心的多视图框架，将个性化网络视图与可迁移的转录组扰动视图对齐，以预测临床药物反应。对于每位患者，我们使用DysRegNet从肿瘤表达构建个性化基因调控网络，并用DrugBank的药物-靶点链接扩展；图神经编码器生成以药物为中心、基于机制的嵌入。同时，一个在LINCS L1000上预训练的冻结条件特异性基因-基因注意力模型为同一患者-药物对生成模拟的扰动后转录组图谱。我们通过带有药物背景困难负例的CLIP风格对比目标在共享潜空间中对齐两个视图，然后连接表示以进行端到端响应分类。在TCGA上，PREDIKTOR在患者、药物和组织分割评估中持续优于最先进基线，并零样本迁移到I-SPY2试验，AUROC比竞争方法提高5.6%。对齐的嵌入产生稳定的基因和通路归因，恢复已知机制，支持可操作和可解释的精准肿瘤学。

## Abstract
Accurate prediction of patient-specific therapeutic response from pre-treatment transcriptomes is hindered by the scarcity of matched clinical response labels and post-treatment molecular profiles. Preclinical transfer-learning models can simulate drug-induced expression changes but are often hard to interpret and unstable, whereas knowledge-graph methods provide mechanistic context yet remain static and fail to capture drug-induced transcriptomic perturbation dynamics. We propose PREDIKTOR, a patient-centered multi-view framework that aligns a personalized network view with a transferable transcriptomic perturbation view to predict clinical drug response. For each patient, we construct an individualized gene regulatory network from tumor expression using DysRegNet and augment it with drug-target links from DrugBank; a graph neural encoder yields a drug-centric, mechanistically grounded embedding. In parallel, a frozen condition-specific gene-gene attention model pretrained on LINCS L1000 generates a simulated post-perturbation transcriptomic profile for the same patient-drug pair. We align the two views in a shared latent space via a CLIP-style contrastive objective with drug-context hard negatives, then concatenate the representations for end-to-end response classification. On TCGA, PREDIKTOR consistently outperforms state-of-the-art baselines under patient-, drug-, and tissue-split evaluations, and transfers zero-shot to the I-SPY2 trial, improving AUROC by 5.6% over competing methods. The aligned embeddings yield stable gene and pathway attributions that recover known mechanisms, supporting actionable and interpretable precision oncology.

## 2. 方法论

### 核心思想
将同一患者-药物对的**两种互补表示**（网络视图 + 扰动视图）通过对齐后拼接，用于端到端药物响应预测。

### 关键技术细节

- **网络视图（Knowledge Graph–based Network View）**
  - 对每位患者，用DysRegNet基于其表达数据构建**个性化基因调控网络**（TF-靶基因），仅保留FDR < 0.05的边缘，使用实验验证的HTRIdb限制候选对。
  - 从DrugBank加入药物-靶点边，形成**异构图**（基因节点+药物节点）。
  - 用DREAMwalk初始化节点嵌入（考虑异质性），再用两层GCN更新。药物节点嵌入经sum-pooling后与患者原始表达（layer-normalized）拼接，得到256维的网络视图表示 \(z_{graph}\)。

- **转录组扰动视图（Transcriptomic Perturbation View）**
  - 条件特异性基因-基因注意力模型（CSG2A）在**LINCS L1000**大规模扰动数据上预训练（冻结），用分子结构（MAT编码）和基因标识作为输入，得到注意力矩阵 \(\alpha_{ij}\)。
  - 对于患者 \(p\) 和药物 \(d\)，用该注意力加权患者基线表达，再通过MLP预测扰动后基因表达，得到978维的扰动视图表示 \(z_{perturb}\)。

- **多视图对齐（Contrastive Alignment）**
  - 两个视图通过线性投影到公共维度，使用**CLIP风格对比损失**最大化正例（同患者同药物）余弦相似度，最小化负例。
  - **关键设计**：负例选为**同一患者不同药物的对**（hard negatives based on common drugs），而非组织类型或标签随机负例，实验证明此策略最优。

- **最终预测**
  - 对齐后的两个表示拼接（256+978=1234维），送入两层MLP（128隐藏层、Dropout、LayerNorm）进行二分类（Sigmoid输出）。
  - 总损失：\(\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{BCE}} + \lambda_{\text{CLIP}} \mathcal{L}_{\text{CLIP-context}}\)，\(\lambda_{\text{CLIP}}\)初始为5，可训练。
- **训练细节**：Adam优化器，学习率1e-4，batch size 32，早停。

## 6. 主要结论与发现

1. PREDIKTOR在**所有三种TCGA分割**和**外部I‑SPY2数据集**上一致超越最先进基线，证明多视图对齐方案的有效性。
2. **两个视图互补**：网络视图提供患者特异性机制背景，扰动视图提供可迁移的动态扰动信号；消融实验显示网络视图贡献略大。
3. **对比学习对齐**显著提升性能，且**硬负例选择（同患者不同药物）**至关重要；去除硬负例或对比学习均导致性能下降。
4. 模型具备**临床可解释性**：基因归因分析富集到已知紫杉醇机制通路（Cell cycle, p53, PI3K-Akt等），且显著富集到LINCS L1000共识紫杉醇上/下调基因。
5. 零样本迁移能力表明方法可在**不同临床情景**（不同平台、不同人群）直接应用。

## 7. 优点（亮点）

1. **新颖的多视图对齐设计**：将个性化图网络与迁移学习扰动模型有机结合，同时解决了静态网络无法捕捉动态和纯数据驱动模型难解释的问题。
2. **精心设计的对比学习策略**：采用药物上下文硬负例（相同患者不同药物），比基于组织或标签的负例更有效，增强模型区分药物特异性效应的能力。
3. **患者特异性个性化网络构建**：使用DysRegNet动态推断每位患者的GRN，而非共享静态网络，更能反映个体异质性。
4. **零样本迁移验证**：在独立临床试验I‑SPY2上直接评估，且提升幅度显著（AUROC +5.6%），增强了临床转化可信度。
5. **强大的可解释性**：不仅能给出预测，还能通过预期梯度归因到基因和通路，且与已有生物学知识一致，支持临床决策。

## 8. 不足与局限

1. **数据集规模小**：TCGA仅包含383对样本、21种药物、358位患者，样本量限制统计功效和泛化评估。且药物种类有限，部分类别（如靶向治疗）代表性不足。
2. **新药泛化困难**：在drug-split上AUROC仅~0.59，远低于patient-split，且网络视图依赖知识图谱中已知药物-靶点关系。对全新/研究性药物（无已知靶点），网络视图失效，模型可能退化为仅依赖扰动视图。
3. **扰动视图表示局限性**：CSG2A基于978个LANDMARK基因且冻结，可能无法完全表征患者特异性扰动效应；药物表示仅用SMILES的MAT编码，未利用3D结构、LLM描述等丰富信息。
4. **计算资源信息不足**：未报告训练时间、GPU数量、模型参数量，影响可复现性和效率评估。
